Every Capbot brain upgrade is signed off-chain via Ed25519 and verified on-chain by the Capmon Anchor program. The receipts below are live, devnet, public — click any signature to inspect the transaction yourself.

What you're looking at: The Capmon staking program enforces tier-locked, monotonic brain progression on Solana. Each row below is a real transaction containing two instructions: an Ed25519SigVerify precompile call signed by the registered upgrade authority, and a capmon-staking::upgrade_brain_v2 instruction that reads the signature out of the instructions sysvar and validates signer + asset + steps + freshness. No admin signature on the transaction itself — the on-chain verifier does all the work. This is Pattern 2 sigverify-ix introspection — and as of this week, every brain upgrade burns 50% of the Capbot's battle winnings as the cost of growth, making each receipt a verifiable proof of burn.
